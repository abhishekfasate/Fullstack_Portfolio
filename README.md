# Portfolio Full-Stack Application

A modern full-stack portfolio application built with **FastAPI**, **React**, and **PostgreSQL**.  
The application showcases projects, blogs, resume, GitHub repositories, and includes analytics tracking with a fully Dockerized setup.

---

## Tech Stack

### Backend
- FastAPI
- Python 3.11
- SQLAlchemy
- PostgreSQL
- Redis
- Celery

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- React Query
- Zustand

### DevOps
- Docker
- Docker Compose
- Nginx
- GitHub Actions

---

## Features

- Responsive portfolio website
- Blog management system
- Project showcase section
- GitHub repository integration
- Resume download support
- Contact form functionality
- Analytics tracking
- Dockerized development and production setup
- CI/CD deployment pipeline
